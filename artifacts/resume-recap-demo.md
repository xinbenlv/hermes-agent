# Resume recap demo artifact

Type: mock demo (renderer-level, not full end-to-end interactive `hermes -c` run)

Why mock:
- This artifact was generated by invoking the resumed-session renderer directly from the test harness.
- It exercises the real formatting code and current implementation, but it is not a full terminal recording of an actual interactive resume session.

Best way to review:
1. Open `artifacts/resume-recap-panel.txt`
2. Compare the panel output against the expected UX goals:
   - short `Summary:` line
   - recent `You:` / `Hermes:` tail
   - continuation-chain note
   - `/history` hint
